# Chicken

Whenever we eat chicken, a variation of this spice mix will be usually used. Perfect for on the fried chicken.

Fry the chicken, and afterwards coat it with this spice mix.

- [ ] 1x        Paprika
- [ ] 1x        Gingerpowder
- [ ] 1x        Freshly grounded peper
- [ ] 1x        Salt
- [ ] 1x        Coriander
- [ ] 1x        Tumuric
- [ ] 0.5x      Smoked paprika
- [ ] 0.5x      Cayenne
- [ ] 0.5x      Fenugreek
- [ ] 0.33x     Cumin
- [ ] 0.25x     Cinnamon
- [ ] 0.25x     Mace
